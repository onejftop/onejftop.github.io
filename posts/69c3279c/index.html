Git 配置

配置个人用户名和邮箱地址

$ git config --global user.name "onejftop"
$ git config --global user.email onejftop@onejftop.com

查看配置信息

查看已有配置信息

$ git config --list

创建本地版本库（仓库）

1.创建新目录并切换目录

$ mkdir 目录名
$ cd 目录名

2.初始化该目录，把把该目录变成Git仓库

$ git init

3.把所有文件添加到仓库

$ git add .

4.把文件提交到仓库

$ git commit -m "本次提交说明"

远程仓库

添加远程库

$ git remote add 远程库别名 远程库链接

推送本地仓库到远程库

$ git push -u 远程库别名 分支

查看当前的远程库

查看当前配置有哪些远程库

$ git remote

查看每个远程库别名的实际链接地址

$ git remote -v

删除远程库

$ git remote rm [远程库别名]

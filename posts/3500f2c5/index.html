多线程实现

继承Thread类

java.lang.Thread 是一个负责线程操作的类，任何类只要继承 Thread 类就可以成为一个线程的主类。线程启动的主方法需要覆写Thread类中的run()方法实现，线程主体类格式的定义如下：

Class 类名 extends Thread{
    属性...	// 类中定义属性
    方法...	// 类中定义方法
    // 覆写Thread类中的run()方法，此方法是线程的主体
    public void run(){
        线程主体方法;
    }
}

实现Runnable接口

Class 类名 implements Runnable{
    @Override
    public void run() {
        
    }
}
// 实例化多线程对象
MyThread mt = new MyThread();
// 利用Thread启动多线程
new Thread(mt).start();
